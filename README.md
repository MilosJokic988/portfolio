# 🕷️ noctis.design - Dark Portfolio

**Enter the dark. Embrace the glitch. Reveal the craft.**

Welcome to my personal portfolio — a collection of eerie, glitchy, and atmospheric web experiments crafted in the shadows. This space is dedicated to exploring the darker side of web design, where style meets chaos and every pixel whispers a story.

## 🩸 About

This portfolio features several interactive websites, each with its own theme — from gothic horrors to mythological depths. All projects are built with **React + Vite**, enhanced with **custom animations**, **creepy sound design**, and **glitch effects**.

## 🕯️ Technologies Used

- ⚛️ React
- ⚡ Vite
- 🎨 TailwindCSS
- 🎭 Framer Motion
- 🔊 Howler.js (for audio)
- 🧪 Custom shaders / CSS effects

## 🖤 Projects

Each project is a standalone dark tale. Some highlights include:

- **DarkTales** – Interactive gothic novel experience.
- **Dostoevsky Quotes** – Existential despair in elegant black & white.
- **Slavic Mythology** – Gods and rituals from the forgotten woods.
- **Lilith** – The exiled one, reimagined through symbols and sound.

New projects are added regularly and featured on [TikTok](https://www.tiktok.com/@noctis.design).
